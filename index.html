State of the Internet

The state of web accessibility on the internet in 2023 & the most common issues across websites.

In 2023, only 3% of websites were accessible. That means that out of 1.5B+ websites, only 48,909,677 websites were accessible.

What

What is an accessible website?

Web accessibility means that websites, tools, and technologies are designed and developed so that people with disabilities can use them. More specifically, people should be able to:

  • perceive, understand, navigate, and interact with the Web, and
  • contribute to the Web

Why

Why make accessible websites?

Everyone deserves equal access to the Web. In a world where almost everything is done digitally, making websites accessible means that everyone, regardless of disability, is able to navigate the Web with minimal to no assistance from other humans. Making a website accessible will also make it accessible for alternate forms of navigation such as tab navigation, keyboard navigation, and screen reader navigation, and not just the traditional mouse/track pad navigation.

How

How do you make a website accessible?

  1. Sufficient contrast between text color and background.
  2. Descriptive Alt-Text of images and graphics, especially those that are important to the content.
  3. Include HTML Structures like headings, sections, etc.
  4. Have descriptive buttons. This means that the text of the button makes sense, not just contextual sense.
  5. Make sure all form data is descriptive and has labels.

Provide Options

There are a lot of different visual disabilities, not all of which we can provide access for via the methods above.

For those that need larger text, more contrast, or less contrast, these features can easily be built into the website through a collection of buttons. To provide even more customizable options, websites like Accessibly App provide a widget that can be added to the website, providing a lot more options for accessibility like dyslexic friendly fonts, and reading masks that make the website easier to use for all.

Accessibility is important. It provides everyone the comfort of using the web with ease, knowing that there are options that make navigation easier.
